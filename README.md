# Academic Font: TimesSimSun

### Overview
A customized font file that merges **Times New Roman** (for English) and **SimSun** (for Chinese). Designed to simplify professional academic typesetting.

### Usage
- For academic paper formatting (Word/LaTeX).
- Ensures seamless bilingual typography.
